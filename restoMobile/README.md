##Getting Started
1. Clone this repo

2. Run "npm install -g ionic cordova" (Note: sudo might be required on Mac or Linux)

3. Run "npm install"

4. Run "ionic serve"

5. Goto "http://127.0.0.1:8100/" in a browser to see the mobile application
